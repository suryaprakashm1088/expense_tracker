{# ── Web Login Action Legend ── #}
Web Login Action Buttons
Send OTP
Sends a WhatsApp one-time code so the member can set their own password via /member/request-otp.
Enable / Disable
Toggles web login on or off. The password hash is kept intact so you can re-enable without a new OTP.
Revoke
Fully removes web access and deletes the password. Member must go through OTP flow again to regain access.
Deactivate
Disables the member from using the WhatsApp bot. Doesn't affect web login.
